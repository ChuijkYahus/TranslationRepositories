# ME Oversize Interface

<Row gap="20">
<BlockImage id="extendedae:oversize_interface" scale="8"></BlockImage>
<GameScene zoom="8" background="transparent">
  <ImportStructure src="../structure/cable_oversize_interface.snbt"></ImportStructure>
</GameScene>
</Row>

ME Oversize Interface has the same number of slots as <ItemLink id="extendedae:ex_interface" />, but it can hold 16 times as many
items per slot (configurable). This means it can hold 1024 (64 x 16) items per slot.

Certain items from other mods can have a specific multiplier configured.
