# Config Modifier

Config Modifier is a tool for bulk modification of configuration inventories.

<ItemImage id="extendedae:config_modifier" scale="4"></ItemImage>

Right-click it to open its GUI.

## Usage

Config Modifier can quickly modify configuration inventories, such as an Interface's, according to its settings; for example,
by setting configured item quantities to the maximum value or removing them all.

You can click a target device (block or subpart) in the world with Config Modifier to modify its configuration.

## Settings

It has two main settings, the **modification mode** and **modification amount (X)**.

You can click a button to change the mode.

![GUI](../pic/cm.png)

- Add/Subtract/Multiply/Divide: Add, subtract, multiply, or divide the configured amount by X.
- Maximize: Set the configured amount to the maximum value.
- Minimize: Set the configured amount to the minimum value.
- Set: Set the configured amount to X.
- Clear: Clear all configured items.
