# Extended Inscriber

<Row gap="20">
<BlockImage id="extendedae:ex_inscriber" scale="8"></BlockImage>
</Row>

Extended Inscriber is an advanced <ItemLink id="ae2:inscriber" />.

It can run 4 inscribing jobs at the same time.

There is a button that allows you to change its inventory's maximum stack size, as with the normal Inscriber.

It is recommended that you set the stack size to 1 when using it with <ItemLink id="ae2:pattern_provider" /> to avoid possible issues.
